progress(无状态进度框)

progress

一个简单的无状态进度指示器

API

/**
  * 显示
  */
 show()

 /**
  * @param txt 加载的文字
  * @param cancel 点击背景是否消失
  */
 showFull(txt，cancel)


/**
 *关闭
 */
 dismiss()

Demo
